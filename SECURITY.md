# Política de Seguridad

## Reportar una vulnerabilidad

Si descubres una vulnerabilidad de seguridad en este proyecto, por favor NO abras un issue público. En su lugar, envía un correo electrónico a [tu-correo@ejemplo.com] (o el medio de contacto que prefieras).

Todas las vulnerabilidades de seguridad serán atendidas con prontitud.

## Versiones Soportadas

Actualmente se brinda soporte de seguridad a las siguientes versiones:

| Versión | Soportada          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |
